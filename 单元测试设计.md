# 大巴扎工具 - 单元测试设计

## 项目概述

基于 Vue 3 + Vue Router + Pinia 的巴扎数据查询应用。6 个路由页面，2 个 Store，3 个 Composable。

## 测试校验流程

1. `npm run test` 执行全部测试（vitest run）
2. 测试环境：happy-dom（模拟 DOM）
3. 页面测试使用 @vue/test-utils 挂载组件，mock fetch 和 router
4. 纯函数测试直接调用 composable 中的函数
5. Store 测试使用 Pinia testing 模式（setActivePinia）
